# Gesundheitsakte 2.0

Neue Hauptversion der bisherigen LaborKompass-App.

Enthalten:
- neuer Name und farbiges Installationssymbol
- intelligenter Import von Labor-, Arzt-, Augenarzt-, Optiker- und Zahnarztdokumenten
- strukturierte Fachwerte, Hilfsmittel und Gesundheitskosten
- anklickbares Körpermodell mit zugeordneten Befunden
- Gesundheitschronik, Laborverläufe und Dokumentvergleiche
- sechs Analyseformen: einfach, ausführlich, erweitert, medizinisch, Arztform und kompaktes Arztgespräch
- erweiterte Analyse mit Symptomen, möglichen Ursachen, Varianten, Alternativen, Vor- und Nachteilen, Gefahren, Gegenmaßnahmen und Warnzeichen
- lokaler Gesundheitsassistent für die eigenen gespeicherten Daten
- lokale und OneDrive-fähige JSON-Sicherung

Wichtig: Die lokale Analyse ist eine Orientierung und keine ärztliche Diagnose. Eine echte externe KI-Auswertung benötigt einen sicheren Proxy-Server. API-Schlüssel niemals in GitHub hochladen.
